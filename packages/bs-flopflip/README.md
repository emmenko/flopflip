# `bs-flopflip`

[BuckleScript](https://github.com/bucklescript/bucklescript) bindings for [flopflip](https://github.com/tdeekens/flopflip) packages - **WIP**

## Install and setup

#### yarn

```bash
$ yarn add bs-flopflip
```

#### bsconfig

Add `bs-flopflip` to your `bs-dependencies`: **bsconfig.json**

```json
"bs-dependencies": [
  "bs-flopflip",
  ...
]
```
